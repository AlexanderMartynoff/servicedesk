Инструкция пользователя

  • 1. Техническая и технологическая поддержка осуществляется в рамках заключенного договора на оказание услуг и при выполнении следующих условий:
    • 1.1 Ответственным лицом со стороны Клиента выполнено требование по постановке задач с указанием:
      • цели, которую необходимо достичь при выполнении заявки;
      • перечня задач или требований (при необходимости предоставить файл с данными или скриншот, иллюстрирующий задачу или требование);
      • ожидаемого результата;
      • дополнительной информации технического и организационного характера, которая может потребоваться для исполнения заявки.
    • 1.2 При заказе услуги ответственным лицом со стороны Клиента, осуществляющий постановку задачи, гарантируется наличие у него последней резервной копии базы данных программы.
    • 1.3 При оформлении заявки на оказание услуг, требующих удаленного подключения к инфраструктуре Клиента, ответственным лицом Клиента (оператором договора) организуется возможность удаленного подключения специалистов ОргЦентра для оказания услуги.
  • 2. Суммарное время работы по заявке с момента ее поступления не регламентируется. Первоначальный срок обработки заявки составляет – 24 часа. В дальнейшем он может быть увеличен в связи с необходимостью проведения дополнительных работ. Специалисты ОргЦентра могут отказать Вам в работах по исполнению заявки, если при выполнении работ потребуется вмешательство на уровне изменения кода конфигурации программы, а также затребовать дополнительные сведения (копии баз данных) для успешного выполнения Вашей заявки.
При невыполнении одного из вышеперечисленных условий Вам может быть отказано в рассмотрении заявки.

REST API naming convention
--------------------------

+ delete (DELETE) - entity/:id
+ create (POST) - entity/
+ update (PUT) - entity
+ get - (GET) entity/:id
+ get - (GET) entity


+ delete (DELETE) - entity/subentity/:id
+ create (POST) - entity/subentity
+ update (PUT) - entity/subentity
+ get - (GET) entity/subentity
+ get - (GET) entity/subentity/:id
